# Basic Library Management System (Java Console)

Minimal single-file Java project to manage books and members.

## Run
1. Save `LibraryApp.java`.
2. Compile: `javac LibraryApp.java`
3. Run: `java LibraryApp`

## Features
- Add/list books
- Borrow/return books
- Add/list members
